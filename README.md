Roadmap:
-Casual HUD (almost done)

-4:3 Support

## Installation
1. Click on 'code' then 'Download Zip'
2. Go to '..\Steam\steamapps\common\Team Fortress 2\tf\custom'
3. Extract the zipped folder in there
4. Enjoy PainHUD
